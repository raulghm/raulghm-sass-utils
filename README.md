@raulghm sass utils and helpers
===================

#### Bower support
```
bower install --save raulghm-sass-utils
```

## Mixins
  * font-size()
  * clearfix()
  * responsive-report()
  * media queries breakpoints
  
## How use

```scss
@import 'bower_components/raulghm-sass-utils/dist/mixins/mixins.scss';

.example {
  @include clearfix();
}
```
